Creates a mixer that blends track mixers.

Parameters
graphThe graph where the mixer playable will be added.
goThe GameObject that requested the graph.
inputCountThe number of inputs on the mixer. There should be an input for each playable from each clip.
Returns
Returns a playable that is used as a mixer. If this method returns Playable.Null, it indicates that a layer mixer is not needed. In this case, a single track mixer blends all playables generated from all layers.

Implemented in UnityEngine.Timeline.AnimationTrack.
